# 🌾 AgroAI — Smart Agriculture System

An AI-powered smart agriculture platform that analyzes soil data, land images, and environmental conditions to recommend the best crops, fertilizers, and farming strategies.

---

## 🚀 Features

* 📷 **Land Image Analysis**

  * Detects soil type using image processing

* 📄 **Soil Report Processing**

  * Extracts NPK, pH, and nutrients from PDF reports

* 📍 **Location Detection**

  * Auto-detects region and season

* 🌱 **Crop Recommendation**

  * Suggests best crops based on soil + climate

* 🧪 **Fertilizer Suggestions**

  * Provides precise fertilizer plans

* 🐛 **Pest Prediction**

  * Identifies possible pest risks and solutions

* 📊 **Analytics Dashboard**

  * Visual charts for soil health and crop performance

## 🧠 How It Works

1. User uploads:

   * Land image
   * Soil report (PDF)

2. System processes:

   * Image → Soil type detection
   * PDF → Nutrient extraction
   * Location → Season detection

3. AI Engine:

   * Compares input with ideal crop conditions
   * Calculates suitability scores

4. Output:

   * Best crops
   * Fertilizer plan
   * Pest risks
   * Farming tips
